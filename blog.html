Keep tabs on how the project is going

Week 3

April 14, 2016

Since the first Origami demo provided by the Holographic Academy, we have tried to download existing piano/keyboard assets online and place them in our project. Unfortunately, we had no luck looking for a reusable asset that suits to the purpose of our application. In the coming week, we will try to build our own custom keyboard asset using Unity cubes.

Although we weren’t able to use the existing assets as they were, we had a chance to experiment with them to figure out how close a user can get to a hologram. Matt’s arms were long enough to touch the keyboards when stretched, but the problem is that no one plays the piano with his/her elbows straight. Would it be possible to induce the users to play the ‘free play’ mode without directly looking at the keyboard?

Plans for the week

  • Field trip to Microsoft!
  • Try to build our own keyboard asset with cubes
  • Resize the custom keyboard asset to the physical keyboard we are using
  • Finalize the features we want in our application
